npm install --global autoprefixer-cli   instalacja pakietu konwertującego css i dodając prefixy wymagane przez niektóre przeglądarki
        ma czytać z pliku style.css i zapisywać wynik do pliku style.prefixed.css (możesz przetestować działanie Autoprefixera np. wpisując w pliku style.css regułę z display: flex),
npm install -g html-validate    sprawdza poprawność kodu html
        ma wyświetlać listę błędów we wszystkich plikach .html.
